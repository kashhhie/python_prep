## üöÄ Quick Start

1. Clone the repo:
```powershell
git clone https://github.com/kashhhie/python_prep.git
```

2. Change into the project folder:
```powershell
cd python_prep
```

3. (Optional) Create and activate a virtual environment:
```powershell
python -m venv .venv; .\.venv\Scripts\Activate.ps1
```

4. Install Jupyter if you don't have it:
```powershell
pip install jupyter
```

5. Start Jupyter Notebook:
```powershell
jupyter notebook
```

6. Open the `python operators` folder and run `operators.ipynb`.
